
# 🛡️ OWASP Top 10 Notes

## 1. Broken Access Control
- Users accessing unauthorized data
- Example: Changing user ID in URL

## 2. Cryptographic Failures
- Weak encryption or no HTTPS

## 3. Injection (SQLi)
- Attacker injects SQL queries
- Example:
  ' OR 1=1 --

## 4. Insecure Design
- Poor application logic

## 5. Security Misconfiguration
- Default credentials, open ports

## 6. Vulnerable Components
- Outdated libraries

## 7. Identification & Authentication Failures
- Weak passwords, no MFA

## 8. Software Integrity Failures
- Untrusted updates

## 9. Logging & Monitoring Failures
- No alert system

## 10. SSRF
- Server making unintended requests
