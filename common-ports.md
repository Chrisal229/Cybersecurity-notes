# 🔌 Common Ports & Protocols

| Port | Protocol | Use |
|-----|---------|-----|
| 80 | HTTP | Web traffic |
| 443 | HTTPS | Secure web |
| 22 | SSH | Remote login |
| 21 | FTP | File transfer |
| 25 | SMTP | Email sending |
| 53 | DNS | Domain resolution |

---

## Key Idea
Ports act as entry points for communication.
